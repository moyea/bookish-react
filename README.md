该项目是基于Acceptance Test驱动的一个react学习性项目。包含e2e和unit测试。

## 如何使用
启动本地json-server

```bash
cd mock-server
node server.js
```

启动项目

```bash
npm run start
```

运行测试

```bash
npm run test
```
